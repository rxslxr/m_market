Вы имеете в виду просто описание “красные пунктирные линии на белом фоне”? Могу привести примеры, как это реализовать в разных контекстах. С чем именно работаете?

Коротко по вариантам:

  • В вебе (CSS/HTML)
    • Для горизонтальной красной пунктирной линии:
      • div.line { border-top: 3px dashed red; height: 0; background: #fff; }
      • или hr { border: 0; border-top: 3px dashed red; height: 0; }
    • Граница вокруг элемента (пунктирный рамка):
      • .box { border: 2px dashed red; background: #fff; }
  • В SVG (масштабируемая графика)
    • <svg width="300" height="6" aria-label="red dashed line"> <line x1="0" y1="3" x2="300" y2="3" stroke="red" stroke-width="2" stroke-dasharray="6 6" /> </svg>
    • Чтобы получить точки (точки вместо длинных отрезков): stroke-dasharray="1 6" и stroke-linecap="round".
  • В Word/PowerPoint
    • Вставка линии, стиль — “пунктирная” и цвет красный; задайте толщину и фон белый.

Если подскажете, где именно нужно реализовать (страница веб, документ, изображение и т. п.), могу дать точный пример или код под вашу задачу.
